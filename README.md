# Overview

This software is a short Java program that displays hello world in both English and Old English.
This software was created to help users learn basic programming concepts in Java by providing a simple 'Hello, World!' application."

[Software Demo Video](https://www.youtube.com/watch?v=xtFoLsgaSyw)

# Development Environment

This software was developed using IntelliJ IDEA 2024.3.5 (Community Edition) on Mac OS Tahoe 26.1.
It uses the Java programming language.
